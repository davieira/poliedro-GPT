Política de Privacidade

Poliedro P+ para ChatGPT / Claude · Última atualização: 9 de junho de 2026

Esta política descreve como o serviço Poliedro P+ para ChatGPT / Claude (API REST e servidor MCP não oficial, código aberto em github.com/davieira/poliedro-GPT) trata informações quando você o utiliza por meio do ChatGPT (Custom GPT com Actions), do Claude (Custom Connector MCP) ou de instalação local (Claude Desktop).

Importante: este projeto não é afiliado ao Poliedro Sistema de Ensino. O portal oficial P+ está em pmais.p4ed.com e possui seus próprios termos.

1. Quem somos

O software é mantido de forma independente por contribuidores do repositório open source poliedro-GPT. Quem faz o deploy da API (por exemplo, no Render) atua como operador técnico da instância que você utiliza.

Dúvidas: GitHub Issues.

2. O que o serviço faz

Consulta, em seu nome, dados do portal Poliedro P+: boletim/notas, mensagens e calendário escolar. Você faz login com usuário e senha do P+; a API valida nos servidores do Poliedro e usa um token temporário para buscar os dados solicitados.

3. Dados que passam pelo serviço

DadoArmazenado?
Usuário e senha do P+Não — só no login, depois descartados
Token JWT do PoliedroSomente em memória (minutos)
Perfil escolar (IDs, matrícula)Cache em memória até 30 min
Notas, mensagens, calendárioNão persistidos — só na resposta
Tokens OAuth (ChatGPT / Claude)Somente em memória, expiração curta

Não há banco de dados. Credenciais do P+ não são gravadas em disco ou logs no servidor.

4. Uso local (Claude Desktop)

Na instalação local, a senha pode ficar no Keychain do macOS — apenas no seu dispositivo, não no servidor remoto.

5. Como os dados são usados

  • Autenticar você no portal Poliedro P+
  • Atender consultas via ChatGPT, Claude ou cliente local
  • Manter sua sessão OAuth enquanto você usa o assistente

Não vendemos nem usamos seus dados para publicidade ou treinamento de modelos de IA.

6. Terceiros

7. Retenção e segurança

Senhas não são retidas. Cache e tokens em memória expiram ou se perdem ao reiniciar o servidor. Comunicação em HTTPS em produção. Cada usuário acessa apenas os próprios dados do P+.

8. Crianças e adolescentes

O serviço pode exibir dados escolares de menores (notas, mensagens, calendário). O acesso deve ser por pais, responsáveis ou o aluno com credenciais válidas do P+.

9. Seus direitos (LGPD)

Você pode solicitar esclarecimentos via GitHub Issues. Para cessar o tratamento, desconecte o Custom GPT ou conector Claude. Como não há banco de dados, exclusão adicional deve ser solicitada ao Poliedro e à OpenAI/Anthropic, conforme as políticas deles.
